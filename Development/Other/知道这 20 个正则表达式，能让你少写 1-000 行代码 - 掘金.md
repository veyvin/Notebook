正则表达式，一个十分古老而又强大的文本处理工具，仅仅用一段非常简短的表达式语句，便能够快速实现一个非常复杂的业务逻辑。熟练地掌握正则表达式的话，能够使你的开发效率得到极大的提升。

正则表达式经常被用于字段或任意字符串的校验，如下面这段校验基本日期格式的JavaScript代码：

``` {style="box-sizing:border-box;outline:none;overflow:auto;font-family:monospace;font-size:1em;line-height:1.3em;"}
var reg = /^(\\d{1,4})(-|\\/)(\\d{1,2})\\2(\\d{1,2})$/; 
var r = fieldValue.match(reg);             
if(r==null)alert('Date format error!');
```

下面是**技匠**整理的，在前端开发中经常使用到的**20**个正则表达式。

#### **1 . 校验密码强度** {#校验密码强度 style="box-sizing:border-box;outline:none;color:rgb(51, 51, 51);"}

密码的强度必须是包含大小写字母和数字的组合，不能使用特殊字符，长度在8-10之间。

``` {style="box-sizing:border-box;outline:none;overflow:auto;font-family:monospace;font-size:1em;line-height:1.3em;"}
^(?=.*\\d)(?=.*[a-z])(?=.*[A-Z]).{8,10}$
```

#### **2. 校验中文** {#校验中文 style="box-sizing:border-box;outline:none;color:rgb(51, 51, 51);"}

字符串仅能是中文。

``` {style="box-sizing:border-box;outline:none;overflow:auto;font-family:monospace;font-size:1em;line-height:1.3em;"}
^[\\u4e00-\\u9fa5]{0,}$
```

#### **3. 由数字、26个英文字母或下划线组成的字符串** {#由数字26个英文字母或下划线组成的字符串 style="box-sizing:border-box;outline:none;color:rgb(51, 51, 51);"}

``` {style="box-sizing:border-box;outline:none;overflow:auto;font-family:monospace;font-size:1em;line-height:1.3em;"}
^\\w+$
```

#### **4. 校验E-Mail 地址** {#校验e-mail-地址 style="box-sizing:border-box;outline:none;color:rgb(51, 51, 51);"}

同密码一样，下面是E-mail地址合规性的正则检查语句。

``` {style="box-sizing:border-box;outline:none;overflow:auto;font-family:monospace;font-size:1em;line-height:1.3em;"}
[\\w!#$%&'*+/=?^_`{|}~-]+(?:\\.[\\w!#$%&'*+/=?^_`{|}~-]+)*@(?:[\\w](?:[\\w-]*[\\w])?\\.)+[\\w](?:[\\w-]*[\\w])?
```

#### **5. 校验身份证号码** {#校验身份证号码 style="box-sizing:border-box;outline:none;color:rgb(51, 51, 51);"}

下面是身份证号码的正则校验。15 或 18位。

15位：

``` {style="box-sizing:border-box;outline:none;overflow:auto;font-family:monospace;font-size:1em;line-height:1.3em;"}
^[1-9]\\d{7}((0\\d)|(1[0-2]))(([0|1|2]\\d)|3[0-1])\\d{3}$
```

18位：

``` {style="box-sizing:border-box;outline:none;overflow:auto;font-family:monospace;font-size:1em;line-height:1.3em;"}
^[1-9]\\d{5}[1-9]\\d{3}((0\\d)|(1[0-2]))(([0|1|2]\\d)|3[0-1])\\d{3}([0-9]|X)$
```

#### **6. 校验日期** {#校验日期 style="box-sizing:border-box;outline:none;color:rgb(51, 51, 51);"}

“yyyy-mm-dd“ 格式的日期校验，已考虑平闰年。

``` {style="box-sizing:border-box;outline:none;overflow:auto;font-family:monospace;font-size:1em;line-height:1.3em;"}
^(?:(?!0000)[0-9]{4}-(?:(?:0[1-9]|1[0-2])-(?:0[1-9]|1[0-9]|2[0-8])|(?:0[13-9]|1[0-2])-(?:29|30)|(?:0[13578]|1[02])-31)|(?:[0-9]{2}(?:0[48]|[2468][048]|[13579][26])|(?:0[48]|[2468][048]|[13579][26])00)-02-29)$
```

#### **7. 校验金额** {#校验金额 style="box-sizing:border-box;outline:none;color:rgb(51, 51, 51);"}

金额校验，精确到2位小数。

``` {style="box-sizing:border-box;outline:none;overflow:auto;font-family:monospace;font-size:1em;line-height:1.3em;"}
^[0-9]+(.[0-9]{2})?$
```

#### **8. 校验手机号** {#校验手机号 style="box-sizing:border-box;outline:none;color:rgb(51, 51, 51);"}

下面是国内 13、15、18开头的手机号正则表达式。

``` {style="box-sizing:border-box;outline:none;overflow:auto;font-family:monospace;font-size:1em;line-height:1.3em;"}
^(13[0-9]|14[5|7]|15[0|1|2|3|5|6|7|8|9]|18[0|1|2|3|5|6|7|8|9])\\d{8}$
```

#### **9. 判断IE的版本** {#判断ie的版本 style="box-sizing:border-box;outline:none;color:rgb(51, 51, 51);"}

IE目前还没被完全取代，很多页面还是需要做版本兼容，下面是IE版本检查的表达式。

``` {style="box-sizing:border-box;outline:none;overflow:auto;font-family:monospace;font-size:1em;line-height:1.3em;"}
^.*MSIE [5-8](?:\\.[0-9]+)?(?!.*Trident\\/[5-9]\\.0).*$
```

#### **10. 校验IP-v4地址** {#校验ip-v4地址 style="box-sizing:border-box;outline:none;color:rgb(51, 51, 51);"}

IP4 正则语句。

``` {style="box-sizing:border-box;outline:none;overflow:auto;font-family:monospace;font-size:1em;line-height:1.3em;"}
\\b(?:(?:25[0-5]|2[0-4][0-9]|[01]?[0-9][0-9]?)\\.){3}(?:25[0-5]|2[0-4][0-9]|[01]?[0-9][0-9]?)\\b
```

#### **11. 校验IP-v6地址** {#校验ip-v6地址 style="box-sizing:border-box;outline:none;color:rgb(51, 51, 51);"}

IP6 正则语句。

``` {style="box-sizing:border-box;outline:none;overflow:auto;font-family:monospace;font-size:1em;line-height:1.3em;"}
(([0-9a-fA-F]{1,4}:){7,7}[0-9a-fA-F]{1,4}|([0-9a-fA-F]{1,4}:){1,7}:|([0-9a-fA-F]{1,4}:){1,6}:[0-9a-fA-F]{1,4}|([0-9a-fA-F]{1,4}:){1,5}(:[0-9a-fA-F]{1,4}){1,2}|([0-9a-fA-F]{1,4}:){1,4}(:[0-9a-fA-F]{1,4}){1,3}|([0-9a-fA-F]{1,4}:){1,3}(:[0-9a-fA-F]{1,4}){1,4}|([0-9a-fA-F]{1,4}:){1,2}(:[0-9a-fA-F]{1,4}){1,5}|[0-9a-fA-F]{1,4}:((:[0-9a-fA-F]{1,4}){1,6})|:((:[0-9a-fA-F]{1,4}){1,7}|:)|fe80:(:[0-9a-fA-F]{0,4}){0,4}%[0-9a-zA-Z]{1,}|::(ffff(:0{1,4}){0,1}:){0,1}((25[0-5]|(2[0-4]|1{0,1}[0-9]){0,1}[0-9])\\.){3,3}(25[0-5]|(2[0-4]|1{0,1}[0-9]){0,1}[0-9])|([0-9a-fA-F]{1,4}:){1,4}:((25[0-5]|(2[0-4]|1{0,1}[0-9]){0,1}[0-9])\\.){3,3}(25[0-5]|(2[0-4]|1{0,1}[0-9]){0,1}[0-9]))
```

#### **12. 检查URL的前缀** {#检查url的前缀 style="box-sizing:border-box;outline:none;color:rgb(51, 51, 51);"}

应用开发中很多时候需要区分请求是HTTPS还是HTTP，通过下面的表达式可以取出一个url的前缀然后再逻辑判断。

``` {style="box-sizing:border-box;outline:none;overflow:auto;font-family:monospace;font-size:1em;line-height:1.3em;"}
if (!s.match(/^[a-zA-Z]+:\\/\\//))
{
    s = 'http://' + s;
}
```

#### **13. 提取URL链接** {#提取url链接 style="box-sizing:border-box;outline:none;color:rgb(51, 51, 51);"}

下面的这个表达式可以筛选出一段文本中的URL。

``` {style="box-sizing:border-box;outline:none;overflow:auto;font-family:monospace;font-size:1em;line-height:1.3em;"}
^(f|ht){1}(tp|tps):\\/\\/([\\w-]+\\.)+[\\w-]+(\\/[\\w- ./?%&=]*)?
```

#### **14. 文件路径及扩展名校验** {#文件路径及扩展名校验 style="box-sizing:border-box;outline:none;color:rgb(51, 51, 51);"}

验证文件路径和扩展名

``` {style="box-sizing:border-box;outline:none;overflow:auto;font-family:monospace;font-size:1em;line-height:1.3em;"}
^([a-zA-Z]\\:|\\\\)\\\\([^\\\\]+\\\\)*[^\\/:*?"<>|]+\\.txt(l)?$
```

#### **15. 提取Color Hex Codes** {#提取color-hex-codes style="box-sizing:border-box;outline:none;color:rgb(51, 51, 51);"}

有时需要抽取网页中的颜色代码，可以使用下面的表达式。

``` {style="box-sizing:border-box;outline:none;overflow:auto;font-family:monospace;font-size:1em;line-height:1.3em;"}
\\#([a-fA-F]|[0-9]){3,6}
```

#### **16. 提取网页图片** {#提取网页图片 style="box-sizing:border-box;outline:none;color:rgb(51, 51, 51);"}

假若你想提取网页中所有图片信息，可以利用下面的表达式。

``` {style="box-sizing:border-box;outline:none;overflow:auto;font-family:monospace;font-size:1em;line-height:1.3em;"}
\\< *[img][^\\>]*[src] *= *[\\"\\']{0,1}([^\\"\\'\\ >]*)
```

#### **17. 提取页面超链接** {#提取页面超链接 style="box-sizing:border-box;outline:none;color:rgb(51, 51, 51);"}

提取html中的超链接。

``` {style="box-sizing:border-box;outline:none;overflow:auto;font-family:monospace;font-size:1em;line-height:1.3em;"}
(<a\\s*(?!.*\\brel=)[^>]*)(href="https?://)((?!(?:(?:www\\.)?'.implode('|(?:www\\.)?', $follow_list).'))[^"]+)"((?!.*\\brel=)[^>]*)(?:[^>]*)>
```

#### **18. 精炼CSS** {#精炼css style="box-sizing:border-box;outline:none;color:rgb(51, 51, 51);"}

通过下面的表达式，可以搜索相同属性值的CSS，从而达到精炼代码的目的。

``` {style="box-sizing:border-box;outline:none;overflow:auto;font-family:monospace;font-size:1em;line-height:1.3em;"}
^\\s*[a-zA-Z\\-]+\\s*[:]{1}\\s[a-zA-Z0-9\\s.#]+[;]{1}
```

#### **19. 抽取注释** {#抽取注释 style="box-sizing:border-box;outline:none;color:rgb(51, 51, 51);"}

如果你需要移除HMTL中的注释，可以使用如下的表达式。

``` {style="box-sizing:border-box;outline:none;overflow:auto;font-family:monospace;font-size:1em;line-height:1.3em;"}
<!--(.*?)-->
```

#### **20. 匹配HTML标签** {#匹配html标签 style="box-sizing:border-box;outline:none;color:rgb(51, 51, 51);"}

通过下面的表达式可以匹配出HTML中的标签。

``` {style="box-sizing:border-box;outline:none;overflow:auto;font-family:monospace;font-size:1em;line-height:1.3em;"}
</?\\w+((\\s+\\w+(\\s*=\\s*(?:".*?"|'.*?'|[\\^'">\\s]+))?)+\\s*|\\s*)/?>
```
