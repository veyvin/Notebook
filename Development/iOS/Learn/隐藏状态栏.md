在iOS7之前可以通过\[\[UIApplication sharedApplication\] setStatusBarHidden:YES\]来实现，iOS7之后有两种方法可以实现

1.重写方法

```objc
//隐藏
- (BOOL)prefersStatusBarHidden {          
    return YES;  
}
//隐藏时的动画
- (UIStatusBarAnimation)preferredStatusBarUpdateAnimation {    
    return UIStatusBarAnimationSlide;
}
```
