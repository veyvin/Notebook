　　其实很简单，用len函数：

&gt;&gt;&gt; array = \[0,1,2,3,4,5\]
&gt;&gt;&gt; print len(array)
6

　　同样，要获取一字符串的长度，也是用这个len函数，包括其他跟长度有关的，都是用这个函数。

　　Python这样处理，如同在print的结果中自动添加一个空格来解脱程序员一样，也是一个人性化的考虑，所以在比如字符串的属性和方法中，就不再用len了，这点要注意一下。
